## 🚀 Como executar

Clone o repositório
```bash
 $ git clone https://github.com/LeoDiasz/college-fullstack-infood
```

Para iniciá-lo, siga os passos abaixo:
```bash
# Instalar as dependências
$ yarn

# Iniciar o projeto
$ yarn start
```

## 🔋 Requisitos Funcionais
 - Realizar criação de usuários
 - Usuário se logar
 - Usuário poder buscar as receitas feitas
 - Usuário poder criar uma propria receita

  
## 📑 Regras de Negócio
- Tem que estar autenticado para criar uma receita
- Não pode criar usuário com mesmo e-mail
